# Twitter Block Search - Chrome Extension

Это Chrome-расширение добавляет кнопку поиска рядом с сообщением "has blocked you" на странице заблокировавшего вас пользователя в Twitter/X. При нажатии открывается **двусторонний поиск** всей переписки между вами и этим пользователем.

## Функциональность

- Автоматически находит текст "@username has blocked you" на странице
- Добавляет элегантную кнопку-лупу рядом с этим текстом
- При клике открывает **двусторонний поиск**: `(from:ваш_ник to:заблокировавший_ник) OR (from:заблокировавший_ник to:ваш_ник)`
- Показывает всю переписку в обе стороны
- Кэширует ваш никнейм для оптимизации производительности
- Открывает результаты поиска в новом окне

## Установка

### Способ 1: Загрузка распакованного расширения

1. Скачайте все файлы плагина в одну папку
2. Откройте Chrome и перейдите в `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами плагина

### Способ 2: Создание иконок

Для полной функциональности создайте иконки:
- `icon16.png` (16x16 пикселей)
- `icon48.png` (48x48 пикселей) 
- `icon128.png` (128x128 пикселей)

Или удалите секцию "icons" из `manifest.json`

## Использование

1. Установите плагин
2. Перейдите на страницу пользователя, который вас заблокировал
3. Найдите текст "@username has blocked you"
4. Нажмите на появившуюся кнопку-лупу
5. Откроется новое окно с поиском ваших сообщений к этому пользователю

## Как это работает

1. **Поиск текста**: Расширение ищет элементы, содержащие "has blocked you"
2. **Извлечение имени**: Извлекает имя заблокировавшего пользователя из текста
3. **Определение вашего ника**: Автоматически определяет ваш ник с кэшированием (5 минут)
4. **Формирование двустороннего запроса**: Создает поиск в обе стороны
5. **Открытие поиска**: Открывает результаты в новом окне

## Технические особенности

- Оптимизированный MutationObserver с debouncing (300ms)
- Кэширование никнейма для повышения производительности
- Поддерживает как Twitter.com, так и X.com
- Адаптируется к светлой и темной теме
- Элегантные анимации и hover-эффекты
- Минимальное потребление ресурсов

## Устранение неполадок

**Плагин не работает:**
- Убедитесь, что вы авторизованы в Twitter/X
- Проверьте, что плагин включен в `chrome://extensions/`
- Обновите страницу после установки

**Не определяется ваш ник:**
- Плагин пытается найти ваш ник в различных местах
- Если не получается, попробуйте перейти на свою страницу профиля
