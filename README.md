# Twitter Block Search

Мини‑расширение для X/Twitter: на странице профиля пользователя, который вас заблокировал, ищет вашу переписку и даёт быстрый переход к найденным сообщениям.

## Что делает

- При открытии профиля блокирующего пользователя запускает поиск ваших сообщений в переписке.
- Во время поиска показывает мигающую лупу.
- Если сообщения найдены — лупа остаётся, по ней можно перейти к переписке.
- Если сообщений нет — выводит клоуна, по нему тоже можно перейти к переписке.

## Установка

1. Скачайте последний `zip`‑архив с расширением.
2. Распакуйте архив в отдельную папку.
3. Откройте страницу расширений в Chrome (`chrome://extensions`) и включите **Режим разработчика**.
4. Перетащите всю папку расширения на страницу расширений — установка завершится автоматически.

**Важно:** не удаляйте установленную папку расширения — Chrome использует её напрямую.

## Как пользоваться

1. Откройте профиль пользователя, который вас заблокировал.
2. Убедитесь, что идёт поиск (мигающая лупа).
3. По завершении нажмите на лупу (если сообщения есть) или на клоуна (если сообщений нет), чтобы перейти к переписке.
