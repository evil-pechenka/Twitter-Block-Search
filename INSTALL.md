# 📦 Установка плагина Twitter Block Search

## Шаг 1: Подготовка файлов

1. Убедитесь, что у вас есть все необходимые файлы:
   ```
   twitter-block-search/
   ├── manifest.json
   ├── content.js
   ├── styles.css
   ├── README.md
   ├── test.html
   └── icon.svg
   ```

2. Создайте папку для плагина на вашем компьютере
3. Скопируйте все файлы в эту папку

## Шаг 2: Установка в Chrome

### Метод 1: Загрузка распакованного расширения

1. Откройте браузер Chrome
2. Введите в адресной строке: `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите кнопку "Загрузить распакованное расширение"
5. Выберите папку с файлами плагина
6. Нажмите "Выбор папки"

### Метод 2: Через меню

1. Откройте Chrome
2. Нажмите на три точки в правом верхнем углу
3. Выберите "Дополнительные инструменты" → "Расширения"
4. Включите "Режим разработчика"
5. Нажмите "Загрузить распакованное расширение"
6. Выберите папку с плагином

## Шаг 3: Проверка установки

1. После установки вы увидите плагин в списке расширений
2. Убедитесь, что переключатель рядом с плагином включен
3. Плагин должен быть активен

## Шаг 4: Тестирование

### Тест на тестовой странице

1. Откройте файл `test.html` в браузере
2. Вы должны увидеть кнопки-лупы рядом с текстами блокировки
3. Нажмите на кнопку и проверьте, что открывается правильный URL

### Тест на реальном Twitter/X

1. Перейдите на страницу пользователя, который вас заблокировал
2. Найдите текст "@username has blocked you"
3. Рядом должна появиться кнопка-лупа
4. Нажмите на кнопку для поиска ваших сообщений

## Устранение неполадок

### Плагин не появляется в списке

- Убедитесь, что все файлы находятся в одной папке
- Проверьте, что файл `manifest.json` не поврежден
- Попробуйте перезапустить Chrome

### Кнопки не появляются

- Убедитесь, что плагин включен в `chrome://extensions/`
- Обновите страницу (F5)
- Проверьте консоль браузера на наличие ошибок (F12)

### Неправильный URL поиска

- Убедитесь, что вы авторизованы в Twitter/X
- Проверьте, что плагин может определить ваше имя пользователя
- Попробуйте перейти на свою страницу профиля

### Ошибки в консоли

1. Откройте DevTools (F12)
2. Перейдите на вкладку Console
3. Ищите ошибки, связанные с плагином
4. Попробуйте переустановить плагин

## Обновление плагина

1. Внесите изменения в файлы плагина
2. Перейдите в `chrome://extensions/`
3. Нажмите кнопку обновления (🔄) рядом с плагином
4. Или перезагрузите плагин вручную

## Удаление плагина

1. Перейдите в `chrome://extensions/`
2. Найдите плагин в списке
3. Нажмите "Удалить"
4. Подтвердите удаление

## Безопасность

⚠️ **Важно**: Этот плагин работает только на сайтах Twitter/X и не собирает никаких данных. Однако используйте его на свой страх и риск.

## Поддержка

Если у вас возникли проблемы:

1. Проверьте, что все файлы на месте
2. Убедитесь, что Chrome обновлен до последней версии
3. Попробуйте отключить другие плагины
4. Переустановите плагин
